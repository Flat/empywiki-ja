## ディストリビューション特定説明書

### Debian 8と上

pymongoバージョン2.8の上が使いできません。

追加的な依存関係:

* ```python3-cffi```

pip使いなら:

* パッケージをインストール ```python3-pip```
* ```pip3 install -r requirements.txt``` を実行します 
* pymongoのバージョン2.8に下げる ```pip3 install pymongo==2.8```を実行します
